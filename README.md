##  Обзор проекта

**Цель:** Создать единую экосистему для командной разработки с:
- Управлением задачами (Kanban)
- Хранением документации
- Интеграцией с Git
- Командным чатом

**Стек технологий:**
- Backend: NestJS, Prisma, PostgreSQL
- Frontend: Next.js, TailwindCSS
- Инфраструктура: Docker, GitHub Actions

##  Архитектура

```mermaid
graph TD
    A[Frontend] --> B[API Gateway]
    B --> C[Auth Service]
    B --> D[Task Service]
    B --> E[Document Service]
```

Ключевые компоненты:

Микросервисная архитектура

JWT-аутентификация

WebSocket для чата

 Быстрый старт
Предварительные требования:
Node.js v18+

Docker

PostgreSQL

Установка:
```bash
git clone https://github.com/zerobug-studio/core.git
cd core
cp .env.example .env
docker-compose up -d
```
